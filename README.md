Лабораторна робота №6
Застосування numpy
---
Мета роботи
Отримати практичні навички роботи з бібліотекою numpy та дослідити методи лінійної регресії: метод найменших квадратів і градієнтний спуск.
---
Опис роботи
У роботі було згенеровано набір двовимірних даних (x, y), які розподілені навколо заданої прямої з додаванням шуму. Це дозволило перевірити точність побудованих моделей.

Реалізовано метод найменших квадратів для обчислення параметрів лінійної регресії. Отримані результати порівняно з функцією numpy.polyfit.

Також реалізовано алгоритм градієнтного спуску для знаходження коефіцієнтів прямої. Проведено підбір параметрів навчання та кількості ітерацій.

Результати
Побудовано графіки:
вхідних даних
лінії регресії, отриманої методом найменших квадратів
лінії, отриманої за допомогою numpy
лінії, отриманої методом градієнтного спуску

Додатково побудовано графік залежності помилки від кількості ітерацій.-
---
Висновки
Метод найменших квадратів дає точне рішення одразу без ітерацій. Градієнтний спуск потребує підбору параметрів та більше обчислень, але є універсальним для складніших задач, де аналітичний розв'язок відсутній.
